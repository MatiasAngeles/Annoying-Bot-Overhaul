# Steps of How to Use It:
1) Use commands pip to install three commands called "discord", "tensorflow" and "numpy?"
2) After you installed those commands, put on the very end your token of your bot in bot.py.
3) Don't forget to use the same token in settings.py just to make the token saved.
4) After you're done, press the run button to wait and make sure all the commands are working.
5) Sometimes you have to press the button twice, after they are all done, use the commands after async def [command name]
# And you're done!
You can replace the words with what your bot is.
